Lenz's Law & Direction of Current

Lenz's Law states that the induced current always flows in a direction that opposes the change in flux that caused it, hence the negative sign:

e = −dΦ/dt  ← the negative sign IS Lenz's Law

If flux is increasing, induced current opposes that increase. If flux is decreasing, induced current tries to maintain it. This is energy conservation - you cannot extract electrical energy without doing mechanical work against this opposing force.

  • 0° → 90°: flux decreasing → current flows in one direction
  • 180° → 270°: flux increasing in opposite sense → current reverses
  • This reversal every half-cycle is what makes it alternating current
Angular Frequency ω vs Regular Frequency f

Two ways to describe rotation speed:

f (Hz) - full cycles per second  |  e.g. 50 Hz = 50 rotations/s
ω (rad/s) - radians per second  |  ω = 2πf

One full cycle = 2π radians, so using ω makes the maths clean: sin(ωt) not sin(2πft).

Real-World Applications of AC
Power Generation
Every power station uses rotating generators. The sine wave is a direct physical result of rotation in a magnetic field.
Transformers
Only AC can be stepped up/down by a transformer. National grids use 400,000 V AC to minimise I²R transmission losses.
AC Motors
Induction motors run directly on AC - the rotating magnetic field that drives them is created by the alternating current itself.
Signal Processing
Audio signals, radio waves, and sensor outputs are all AC sine waves. Fourier analysis decomposes any signal into sine components.

Why a Sine Wave?

The coil rotates at constant angular velocity ω. The induced EMF depends on how fast field lines are being cut - which varies sinusoidally:

  • θ = 0° - Coil face ⊥ B → zero cutting → EMF = 0
  • θ = 90° - Coil edge cuts field → max cutting → EMF = +Vm
  • θ = 180° - Face ⊥ B again → zero cutting → EMF = 0
  • θ = 270° - Max cutting, reversed → EMF = −Vm
e(t) = Vm · sin(ωt)  ← because v ⊥ B · sin(θ)
Φ = NBA·cos(ωt)  →  e = −dΦ/dt = NBAω·sin(ωt)

Key Physical Insights
  • At θ = 0° - coil face ⊥ to B → Φ is maximum → e = 0 (no flux cutting)
  • At θ = 90° - coil edge ∥ to B → Φ = 0 → e = +Vm (max flux cutting)
  • At θ = 180° - same geometry as 0° → e = 0 again
  • At θ = 270° - max cutting, reversed direction → e = −Vm
  • The EMF is the rate of change of flux (Faraday's Law), not the flux itself
  • Lenz's Law: induced current opposes the change in flux that caused it
  • AC current reverses direction every half cycle (every 180°)

Core EMF Formulas

Instantaneous EMF:

e(t) = Vm · sin(ωt + φ)

Peak / Maximum EMF:

Vm = N · B · A · ω

Angular Frequency:

ω = 2πf = 2π / T

Flux (Faraday Source):

Φ = N · B · A · cos(ωt)

Faraday's Law:

e = −dΦ/dt

Full derivation:

d/dt[NBA·cos(ωt)] = −NBAω·sin(ωt)
∴ e = NBAω·sin(ωt) = Vm·sin(ωt)
RMS, Average & Factors

RMS Voltage (≡ DC heating):

Vrms = Vm / √2 ≈ 0.707 · Vm

Average Voltage (half-cycle):

Vavg = 2Vm / π ≈ 0.637 · Vm

Crest / Peak Factor:

Kp = Vm / Vrms = √2 ≈ 1.414

Form Factor:

Kf = Vrms / Vavg = π/(2√2) ≈ 1.111

RMS Current:

Irms = Im / √2 = Im · 0.707

Why RMS?

An AC of Vrms produces the same heating effect in a resistor as a DC of the same value. 230V AC mains → Vm = 230 × √2 ≈ 325V peak!

Power Formulas

Instantaneous Power:

p(t) = v(t) · i(t) = Vm·Im·sin²(ωt)

Real (Active) Power:

P = Vrms · Irms · cos(θ) [W]

Reactive Power:

Q = Vrms · Irms · sin(θ) [VAR]

Apparent Power:

S = Vrms · Irms [VA]

Power Triangle:

S² = P² + Q²

Power Factor:

PF = cos(θ) = P / S

Energy:

W = P · t [Joules or kWh]
Impedance (AC Circuits)

Ohm's Law for AC:

V = I · Z  (Z in Ohms)

RLC Impedance:

Z = √(R² + (XL − XC)²)

Inductive Reactance:

XL = ω · L = 2πfL [Ω]

Capacitive Reactance:

XC = 1/(ω·C) = 1/(2πfC) [Ω]

Phase Angle:

θ = arctan((XL − XC) / R)

Phasor Impedance:

Z = R + j(XL − XC)
Resonance & Bandwidth

Resonant Frequency:

f₀ = 1 / (2π√LC) [Hz]

At Resonance (XL = XC):

Z_min = R,  PF = 1,  P = max

Quality Factor Q:

Q = ω₀L / R = 1/(ω₀CR)

Bandwidth:

BW = f₀ / Q [Hz]

Half-power frequencies:

f₁, f₂ = f₀ ± BW/2

At f₁ and f₂: power = half of peak power, |Z| = R√2

Transformer Formulas

Turns Ratio:

a = N₁ / N₂ = V₁ / V₂

Voltage Transform:

V₂ = V₁ · (N₂ / N₁)

Current Transform:

I₂ = I₁ · (N₁ / N₂)

Power Conservation:

V₁ · I₁ = V₂ · I₂ (ideal)

Efficiency:

η = Pout / Pin × 100%

Step-up: N₂ > N₁ → V₂ > V₁ (grid transmission). Step-down: N₂ < N₁ → V₂ < V₁ (household supply).

The War of Currents (1880s)

In the 1880s, Nikola Tesla (AC) and Thomas Edison (DC) competed to establish the dominant electrical standard. Tesla won with one critical insight:

P_loss = I² · R

By stepping voltage up with a transformer, current decreases proportionally - slashing transmission losses. A transformer cannot work on DC.

  • AC can travel hundreds of kilometres at high voltage
  • Edison's DC stations needed one every 1.6 km (1 mile)
  • Today, HVDC (High Voltage DC) is used for very long submarine or underground cables where AC losses would be too high
  • Modern grids use AC backbone with DC conversion at endpoints

RLC Theory

In an RLC series circuit, all three elements share the same current but have different voltage relationships:

  • VR - in phase with current (resistor)
  • VL - leads current by 90° (inductor)
  • VC - lags current by 90° (capacitor)
VS = √(VR² + (VL − VC)²)

At resonance (f₀), XL = XC, they cancel perfectly. Current is maximum, impedance equals just R, power factor = 1.

f₀ = 1 / (2π√LC)
